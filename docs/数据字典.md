# 数据字典

## user（用户）

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER | 主键 |
| username | TEXT | 用户名（唯一） |
| password | TEXT | 密码（明文存储） |
| permission_level | INTEGER | 权限等级：1=查看、2=编辑、3=管理 |
| created_at | DATETIME | 创建时间 |

## employee（员工/经手人）

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER | 主键 |
| name | TEXT | 姓名（唯一） |
| department | TEXT | 部门 |
| phone | TEXT | 电话 |
| remark | TEXT | 备注 |
| created_at | DATETIME | 创建时间 |

## material_category（物料分类）

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER | 主键 |
| code | TEXT | 分类编码（唯一），2位大类 + 2位小类 |
| name | TEXT | 分类名称 |
| parent_code | TEXT | 父级分类编码，大类为 NULL |
| level | INTEGER | 层级：1=大类、2=小类 |

## material（物料）

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER | 主键 |
| code | TEXT | 物料编码（唯一） |
| name | TEXT | 物料名称 |
| spec | TEXT | 规格型号 |
| unit | TEXT | 单位 |
| category_code | TEXT | 分类编码 |
| manufacturer | TEXT | 品牌/供应商 |
| storage_condition | TEXT | 存储条件，默认"常温" |
| shelf_life | INTEGER | 保质期（天） |
| remark | TEXT | 备注 |
| is_reusable | INTEGER | 是否可回用：0=否、1=是 |
| safety_stock | REAL | 安全库存量 |
| created_at | DATETIME | 创建时间 |

## supplier（供应商）

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER | 主键 |
| name | TEXT | 供应商名称 |
| contact | TEXT | 联系人 |
| phone | TEXT | 联系电话 |
| address | TEXT | 地址 |
| created_at | DATETIME | 创建时间 |

## in_order（入库单主表）

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER | 主键 |
| order_no | TEXT | 入库单号（唯一） |
| supplier_id | INTEGER | 供应商 ID → supplier.id |
| operator_id | INTEGER | 操作员 ID → user.id |
| status | TEXT | 状态：pending=待审核、approved=已审核、cancelled=已取消 |
| remark | TEXT | 备注 |
| receiver | TEXT | 经手人 |
| purpose | TEXT | 用途 |
| receiver_date | DATE | 入库日期 |
| created_at | DATETIME | 创建时间 |
| approved_at | DATETIME | 审核时间 |
| approved_by | INTEGER | 审核人 ID → user.id |

## in_order_item（入库单明细）

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER | 主键 |
| order_id | INTEGER | 入库单 ID → in_order.id |
| material_id | INTEGER | 物料 ID → material.id |
| batch_no | TEXT | 批次号 |
| production_date | DATE | 生产日期 |
| expiry_date | DATE | 有效日期 |
| quantity | REAL | 数量 |
| unit_price | REAL | 单价 |
| remark | TEXT | 备注 |

## out_order（出库单主表）

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER | 主键 |
| order_no | TEXT | 出库单号（唯一） |
| operator_id | INTEGER | 操作员 ID → user.id |
| status | TEXT | 状态：pending=待审核、approved=已审核、completed=已完成 |
| remark | TEXT | 备注 |
| purpose | TEXT | 用途 |
| created_at | DATETIME | 创建时间 |
| approved_at | DATETIME | 审核时间 |
| approved_by | INTEGER | 审核人 ID → user.id |
| department | TEXT | 部门 |
| receiver | TEXT | 领用人 |
| receiver_date | DATE | 出库日期 |

## out_order_item（出库单明细）

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER | 主键 |
| order_id | INTEGER | 出库单 ID → out_order.id |
| material_id | INTEGER | 物料 ID → material.id |
| batch_no | TEXT | 批次号 |
| unit_price | REAL | 单价 |
| remark | TEXT | 备注 |
| requested_quantity | REAL | 申请用量 |
| actual_quantity | REAL | 实际用量 |
| returned_quantity | REAL | 已退回数量 |
| initial_gross_weight | REAL | 领用毛重（kg，仅可回用物料） |
| shipment_info | TEXT | 实发信息（仅可回用物料） |

## inventory（库存）

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER | 主键 |
| material_id | INTEGER | 物料 ID → material.id |
| batch_no | TEXT | 批次号 |
| production_date | DATE | 生产日期 |
| expiry_date | DATE | 有效日期 |
| quantity | REAL | 当前库存数量 |
| in_order_item_id | INTEGER | 来源入库明细 ID → in_order_item.id |
| updated_at | DATETIME | 更新时间 |

唯一约束：`(material_id, batch_no)`

## return_order（退库单主表）

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER | 主键 |
| order_no | TEXT | 退库单号（唯一） |
| related_out_order_id | INTEGER | 关联出库单 ID → out_order.id |
| department | TEXT | 部门 |
| receiver | TEXT | 退库人 |
| receiver_date | DATE | 退库日期 |
| operator_id | INTEGER | 操作员 ID → user.id |
| status | TEXT | 状态：pending=待审核、approved=已审核 |
| remark | TEXT | 备注 |
| created_at | DATETIME | 创建时间 |
| approved_at | DATETIME | 审核时间 |
| approved_by | INTEGER | 审核人 ID → user.id |

## return_order_item（退库单明细）

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER | 主键 |
| return_order_id | INTEGER | 退库单 ID → return_order.id |
| out_order_item_id | INTEGER | 出库明细 ID → out_order_item.id |
| material_id | INTEGER | 物料 ID → material.id |
| batch_no | TEXT | 批次号 |
| remark | TEXT | 备注 |
| return_gross_weight | REAL | 退回毛重（kg） |
| actual_net_weight | REAL | 实际净用量（kg） |

## reusable_material_weight（可回用物料称重记录）

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER | 主键 |
| out_order_item_id | INTEGER | 出库明细 ID → out_order_item.id（唯一） |
| material_id | INTEGER | 物料 ID → material.id |
| initial_gross_weight | REAL | 领用毛重（kg） |
| initial_weight_time | DATETIME | 领用称重时间 |
| initial_operator_id | INTEGER | 领用称重操作员 → user.id |
| return_gross_weight | REAL | 退回毛重（kg） |
| return_weight_time | DATETIME | 退回称重时间 |
| return_operator_id | INTEGER | 退回称重操作员 → user.id |
| actual_net_weight | REAL | 实际净用量（kg） |
| status | TEXT | 状态：checked_out=已领出、returned=已退回 |
| remark | TEXT | 备注 |

## 索引

| 表 | 索引列 | 目的 |
|----|--------|------|
| in_order | order_no | 入库单号快速查找 |
| out_order | order_no | 出库单号快速查找 |
| inventory | material_id | 库存按物料查询 |
| return_order | order_no | 退库单号快速查找 |
| return_order | related_out_order_id | 按关联出库单查找退库 |
